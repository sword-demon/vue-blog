    Go结构体和接口

    # 结构体

    结构体是一个复合类型，用于表示一组数据。

    结构体由一系列属性组成，每个属性都有自己的类型和值

    // 定义一个结构体
type Person struct {
  name string
  age int
  email string
}

// 初始化结构体
var p1 = Person{"wujie", 19, "wyujui@qq.com"}

// 结构体中取值
fmt.Println(p1.name, p1.age, p1.email)

p1.age = 20 // 修改值
